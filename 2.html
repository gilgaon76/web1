CSS

Cascading Style Sheets(CSS) is a style sheet language used for describing the presentaion of a document written in a markup language. Although most often used to set the visual style of web pages and user interfaces written in HTML and XHTML, the language can be applied to any XHL document, including plain XML, SVG and XUL, and is applicable to rendering in speech, of o other media. Along with HTML and used by most websites to create visually engaging webpages, user interfaces for web applicaions, and user intergaces for many mobile applications.
